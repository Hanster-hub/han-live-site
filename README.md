# han-live-site

Personal portfolio site — plain HTML, CSS, and a little vanilla JS. No frameworks, no build step.

Live at: https://hanster-hub.github.io/han-live-site/

## Structure

```
public/   ← all site files (HTML, CSS, JS, assets)
```

## Deploy

Push to `main`. GitHub Actions publishes the `public/` directory to GitHub Pages automatically.
